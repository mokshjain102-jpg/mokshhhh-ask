# 🌱 AgroVision AI

AI-Powered Crop Disease Detection & Smart Farming Assistant

---

## 🚀 Overview
AgroVision is an AI-based platform that helps farmers detect crop diseases early using image input. It also provides treatment suggestions, prevention tips, and smart farming insights.

---

## 🔥 Features

- 🌿 Disease Detection using AI  
- 📊 Severity Analysis  
- 🌦 Weather Risk Prediction  
- 💊 Smart Treatment Suggestions  
- 📸 Live Demo (Upload Image)  
- 📊 Dashboard (Scan Data Insights)  
- 🚜 Farmer Help Mode  
- 📍 Location-Based Advisory  

---

## 🧠 How It Works

1. Upload crop leaf image  
2. AI analyzes disease  
3. Shows:
   - Disease Name  
   - Confidence Score  
   - Severity  
   - Treatment & Prevention
